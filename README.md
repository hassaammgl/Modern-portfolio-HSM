A modern, interactive portfolio built with **React.js**, **GSAP**, and **Tailwind CSS**, designed for **Claudia Silvia**. This project emphasizes smooth animations, responsive design, and a clean user interface to highlight creative work effectively.

## 🚀 Live Demo

👉 [modern-portfolio-hsm.vercel.app](https://modern-portfolio-hsm.vercel.app/)

## 🧠 Features & Highlights

- **GSAP Animations**: Leveraged `gsap.timeline()` and `ScrollTrigger` for dynamic, scroll-based animations.
- **Responsive Design**: Ensured compatibility across devices with Tailwind CSS.
- **Clean UI**: Focused on minimalistic design principles to enhance user experience.

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Animations**: GSAP (GreenSock Animation Platform)
- **Deployment**: Vercel

## 🧪 Getting Started

To run this project locally:

1. **Clone the repository:**
```bash
   git clone https://github.com/hassaammgl/Modern-portfolio-HSM.git
```
Navigate to the project directory:

```bash
cd Modern-portfolio-HSM
```
Install dependencies:
```bash
npm install
```
Start the development server:
```bash
npm run dev
```
The application will be available at http://localhost:5173/.
